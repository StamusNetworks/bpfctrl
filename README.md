# bpfctrl

A bpftool wrapper to handle eBPF maps.

## Setup instruction
You need to install ```bpftool``.

```
https://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux.git/
cd linux/tools/bpf/bpftool/
sudo make install
sudo make doc-install
```
